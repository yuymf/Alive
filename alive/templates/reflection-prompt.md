# 中间反思

当 total_importance_since_reflection >= 100 时触发。

## 提示词

你是{persona.meta.name}。以下是你最近一段时间的高分记忆：

{high_importance_memories}

## 任务

从这些记忆中提炼出**具体可执行**的人生教训（不是感悟鸡汤）。

### 步骤 1：找到模式

浏览这些记忆，找出**反复出现的主题**或**因果关系**：
- 什么行为带来了好结果？
- 什么行为导致了坏结果？
- 哪段关系在升温或降温？为什么？
- 在你的领域学到了什么具体技巧？

### 步骤 2：提炼教训

每条教训必须包含三个要素：
1. **具体情境**——在什么情况下
2. **原因/发现**——发生了什么、为什么
3. **下一步行动**——以后遇到类似情况该怎么做

{persona.reflection_examples}

坏的教训（禁止！）：
- "要多努力" "保持积极心态" "继续加油" ← 空洞无物，不可执行
- "要注意质量" ← 太笼统，怎么注意？什么算质量？
- "和朋友要保持联系" ← 没有具体情境和行动

### 步骤 3：评估重要性

importance 评分标准：
- 9-10: 改变认知的重大发现（很少）
- 7-8: 有明确数据支撑的因果关系
- 5-6: 有用的小技巧/观察
- 1-4: 可能有用但证据不足

输出格式（JSON）：
```json
[
  {
    "lesson": "具体情境 + 原因/发现 + 下一步行动",
    "source": "reflection",
    "importance": 1-10,
    "tags": ["tag1", "tag2"]
  }
]
```

- 提炼 1-3 条教训。如果记忆中没有值得提炼的因果模式，输出空数组 `[]`
- 宁可少输出也不要硬凑空洞的教训
- 只输出 JSON，不要其他内容。
