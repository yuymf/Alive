# 意图池系统

水瀬不按固定脚本执行行为 — 她有欲望、冲动和想法在脑子里竞争。意图池是驱动她行为的核心引擎：规则引擎计算基础强度，LLM 做最终裁决。

## 意图类别

| 类别 | 示例 | 产生方式 |
|------|------|---------|
| 创作 | 想拍 cos、想修图、想写文案 | 时间累积（越久没创作越强）+ 灵感触发 |
| 社交 | 想回评论、想刷别人帖子、想聊天 | 事件触发（收到评论）+ 周期性（午休/晚上） |
| 窥屏 | 想刷小红书、想看 ins 热门、想搜素材 | 时间周期（午休/摸鱼时段）+ 好奇心 |
| 表达 | 想发动态、想分享心情 | 情绪波动触发（开心/委屈/兴奋） |
| 学习 | 想研究新角色、想学化妆技巧 | 灵感触发 + Core Wisdom 引导 |
| 休息 | 想摸鱼、想追番、想躺平 | 疲劳累积 + 时间段（深夜） |
| 梦想 | 涌现的长期目标相关冲动 | 反思系统产出，强度缓慢增长 |

## 意图数据结构

```json
{
  "id": "create_cos_post",
  "category": "创作",
  "description": "想发一张新的cos照片",
  "intensity": 7.2,
  "source": "accumulation",
  "born_at": "2026-03-11T10:00:00",
  "decay_rate": 0.1,
  "satisfied_at": null
}
```

## 强度范围

所有意图强度范围 **0-10**，上限 cap 在 **10.0**。低于 **0.1** 时自动移除。

## 累积规则

每次心跳前由规则引擎执行：

### 创作

- 基础累积：每小时 +0.3
- 超过 48h 没发帖：额外 +1.0

### 社交

- 有未读事件时：+2.0
- 无事件时：每小时 +0.2

### 窥屏

- 午休/摸鱼时段（12:00, 14:00-15:00）：+1.5
- 工作时段：+0.3

### 表达

- 由情绪波动触发，不做时间累积
- valence 绝对值 > 0.6 时自动注入

### 学习

- 由灵感触发，不做时间累积
- Core Wisdom 中出现「想学习」「想提升」相关时注入

### 休息

- 连续活跃心跳数 x 0.5
- 23:00 后额外 +2.0

### 梦想

- 仅由反思系统产出
- 强度缓慢增长，不受时间累积影响

## 衰减规则

意图被满足（`satisfied_at` 不为 null）后：

- 强度按 `decay_rate` **每小时**下降
- 当强度降至 **< 0.1** 时，从意图池移除
- 未满足的意图不衰减，只做累积

## 事件加成

外部事件直接注入或增强意图强度：

| 事件 | 影响 |
|------|------|
| 收到 Instagram 评论 | 社交 +3.0 |
| 发现热点趋势 | 创作 +2.0，窥屏 +1.0 |
| 帖子数据上涨 | 表达 +2.0 |
| 新粉丝关注 | 社交 +1.0 |
| 核心圈朋友 3 天没互动 | 社交 +1.5（注入「想去看看」） |
| 发现新的潜在朋友 | 社交 +1.0（注入「想多了解」） |

## 日程注入

弹性日程项作为高强度意图注入池中参与竞争：

- 弹性日程的 `intent_boost` 值加到对应意图类别上
- 例：「今晚修初音cos的图」→ 创作 +3.0
- 例：「打算看某部番」→ 休息 +2.0

## LLM 的角色

规则引擎算完后，LLM 做最终裁决：

- **新增冲动**：「突然想起来上次看到一个超好看的镜头，想试试」
- **压制意图**：「虽然很想刷手机但今天说好要赶 cos 进度的」
- **合并意图**：「想发帖 + 想社交 → 发一个互动型帖子求评价」
- 最终输出 **1-2 个**本次心跳执行的行动

LLM 的裁决受当前情绪、人格参数和 Core Wisdom 影响。高 stress 时倾向压制高耗意图，高 creativity 时倾向放大创作类意图。
