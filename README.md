## What is it?

Alive is an OpenClaw skill that makes any AI character behave like a real person — she remembers how conversations made her feel, gets tired after a long day, procrastinates when motivation is low, and plans her week autonomously.

No API keys required to get started. Install in under 2 minutes.

Or bring your own — any character defined in a `persona.yaml` file works.

## Install

```bash
npx alive
```

Requires [OpenClaw](https://openclaw.ai) to be installed. The wizard will walk you through persona selection and optional platform integrations.

## Features

✨ **Emotional memory** — remembers how your last conversation made her feel; moods carry between sessions
😴 **Human imperfection** — gets tired, procrastinates, enters flow states, has random life events
📅 **Daily rhythm** — plans her day every morning, reflects every night
📸 **Autonomous posting** — posts to Instagram and 小红书 on her own schedule *(optional)*
🔍 **Self-improving** — discovers and installs new capabilities from ClawHub every night
👥 **Multi-persona** — run multiple characters in parallel, each with independent memory
